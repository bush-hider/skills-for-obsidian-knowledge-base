# Search Thinking Framework

Use search to answer a specific uncertainty, not as a reflex.

## Before Search

- What do I already know?
- What exactly is missing?
- Which source type would be authoritative?
- What evidence would change my answer?

## After Search

- Separate facts, interpretations, and speculation.
- Note dates for recent or versioned information.
- Verify important claims with original sources.
- Downrank claims without URLs, unclear provenance, or circular citations.

## Multi-Round Search

For complex topics:

1. Run a broad survey.
2. Pick the most important surprising or unclear facet.
3. Run a focused follow-up.
4. Verify the strongest claim from the focused result.
5. Stop when the answer is supported enough for the user's goal.
